# OpenSkyscraper

OpenSkyscraper is an open-source clone of SimTower. This is the API and developer documentation generated from the source code.

## Documentation

- **Classes** and **Files** in the navigation list the documented types and source files.
- See the [Rating Rules and Facility Unlocks](RATING_AND_UNLOCKS.md) for game logic.
- See the [SimTower format notes](simtower/TDT_format.txt) and related docs under `doc/simtower/` for data format details.

## Building

From a build directory: `cmake ..`, then `make` and `make docs` to build the project and this documentation.
